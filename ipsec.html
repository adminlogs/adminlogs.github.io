IPSec Phase 1

In IPSec phase 1 a tunnel is created used to protect IPSec management traffic. This tunnel is bidirectional. Three steps are needed to create this tunnel.

Step 1: Negotiate the IKE Phase 1 Tunnel

The two routers negotiate the Internet Key Exchange (IKEv1) Phase 1 tunnel. This can be done in one of the two modes, Main mode or Aggreessive Mode. Main mode is more secure than the Aggressive Mode and the Main mode uses more packets thans the Aggressive mode. Most current VPNs use by default the Main mode.

Five basic items need to be agreed upon between the two VPN devices/gateways for the IKE Phase 1 tunnel to succeed, as follows.

  • Hash algorithm
  • Encryption algorithm
  • Authentication method
  • Lifetime
  • Diffie-Hellman (DH) group
